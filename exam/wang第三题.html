In this question, we first let all the observations to be one data set called data and then sort them by using the function sorted_data <- sort(). After that, as the question has mentioned, we consider this as a binomial distribution with a success rate of \(\frac{9}{14}\) and the number of trails is \(7\). So we can calculate the theoretical quantile by qbinom(ppoints(sorted_data), size = n, prob = p). ppoints is a function that generates the loci.

Then, we have all the data we need, plot the empirical quantile against the theoretical one and draw a theoretical line in red using abline(0, 1, col = "red").

As we can see from the Q-Q plot, we can see that the data are not a perfect fit to the given binomial distribution, but there are no extreme systematic deviations: the most significant differences are centered in the two tail ends.

We first set our hypothesis set \(H_0\) as the observed data follows the specified distribution with the given parameters. In other words, the data follows the distribution with PMF \(r(x)\), where the parameters \(a, b, c, d, e, f, g\) represent the probabilities of the corresponding outcomes (with \(a + b + c + d + e + f + g = 1\)). Then null hypothesis \(H_1\) will be the observed data does not follow the specified distribution.

In order to tell whether we can accept the hypothesis \(H_0\) or not, we can use the Chi-square test:

chisq_test <- chisq.test(x = table(data), p = parameters)
## Warning in chisq.test(x = table(data), p = parameters): Chi-squared
## approximation may be incorrect
chisq_test
## 
##  Chi-squared test for given probabilities
## 
## data:  table(data)
## X-squared = 0, df = 6, p-value = 1

The result of the chi-square test gives a p-value = 1 > 0.05, which means we cannot reject the hypothesis \(H_0\), so that this type of distribution describes the observed data well.

set.seed(100)
bootstrap <- function(data, B = 10000){
  bootstrap_samples <- numeric(B)  
  for (i in 1:B){
    sample_data <- sample(data, size = n, replace = TRUE)
    bootstrap_samples[i] <- mean(sample_data)
  }
  return(bootstrap_samples)
}
alpha <- 0.05
CI <- quantile(bootstrap(data, B = 10000), c(alpha/2, 1-alpha/2))
CI
##     2.5%    97.5% 
## 4.550000 5.333333

Here, in order to calculate the confidence interval by using Bootstrap, we first define a function for Bootstrap. As bootstrap includes the sample with replacement of 60 points from data. Then we calculate the bootstrap_samples for each time of sampling and repeat these two steps for 10000 times.

At last, we use percentile interval this time to calculate the CI: \[C_n = \left(\theta^*_{\alpha/2}, \theta^*_{1-\alpha/2}\right) \] So, we can get the upper bound and the lower bound for a confidence interval of 95%: [4.550000, 5.333333].
